# Jungle Safari Experts

Premium Next.js landing page for [Jungle Safari Experts](https://junglesafariexperts.com/) — India's trusted wildlife safari tour specialists.

## Features

- Modern, mobile-first design inspired by premium landing pages
- Signature safari tours: Jhalana, Amargarh, Ranthambore, Sariska
- Contact enquiry form with WhatsApp & call CTAs
- SEO-ready metadata and fast static generation

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Deploy

Deploy easily on [Vercel](https://vercel.com) by connecting this GitHub repository.

## Tech Stack

- Next.js 16
- React 19
- Tailwind CSS 4
- TypeScript
